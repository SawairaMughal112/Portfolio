# Sawaira Mughal - Portfolio Website

A modern, responsive portfolio website built with **Vite**, **React**, and **Tailwind CSS**.

## 🚀 Features

- **Responsive Design** - Fully responsive on mobile, tablet, and desktop
- **Modern Tech Stack** - Built with latest tools and best practices
- **Fast Performance** - Optimized with Vite for quick load times
- **Clean UI** - Professional design using Tailwind CSS
- **SEO Friendly** - Proper meta tags and semantic HTML

## 📋 Sections

- **Hero** - Eye-catching introduction
- **About** - Personal background and contact info
- **Skills** - Frontend, Backend, Databases, and Core Competencies
- **Projects** - Showcase of key projects
- **Experience** - Professional work history
- **Education** - Degrees and courses completed
- **Achievements** - Key accomplishments
- **Contact** - Easy ways to get in touch

## 🛠️ Installation

1. **Clone or download** the project
2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 🎨 Customization

### Colors
- Edit color values in `tailwind.config.js` under theme → colors
- Main colors: primary (#1f2937), accent (#3b82f6)

### Content
- Update personal information in component files (About.jsx, Contact.jsx, etc.)
- Modify projects in Projects.jsx
- Update experience in Experience.jsx

### Styling
- Global styles in `src/index.css`
- Component-specific styles use Tailwind utility classes

## 🚀 Deployment

### Deploy to Netlify

